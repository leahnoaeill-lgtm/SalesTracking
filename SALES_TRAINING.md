# BE Strategy Sales Portal — Training Guide

Welcome! This portal helps the sales team track activities with bronchiectasis (BE) clinics and physicians. This guide walks you through everything you need to know.

**Portal URL:** http://44.251.113.46:5004

---

## 1. Getting Started

### First-time setup
1. Open the portal URL in your browser (Chrome, Edge, or Safari).
2. In the top-right corner of the header, type **your full name** (e.g., `Jane Smith`) into the "Your name" box and click **Save**.
3. Your name is remembered on this browser. It will be attached to every activity and next-step you log, so we know who did what.

> **Important:** You must set your name before you can log activities or update next steps.

---

## 2. The Dashboard (Main Page)

The dashboard shows every physician account with their key information.

### Columns you can see
| Column | What it means |
|---|---|
| Last Name / First Name / Credentials | Physician identity |
| Specialty | Medical specialty (e.g., Pulmonology) |
| Claims / Beneficiaries | 2023 Medicare volume for HCPCS E0483 |
| City / State | Practice location |
| KAM | Key Account Manager assigned (Mark Franzen = West, Mike Grillo = East) |
| Clinic Name | Primary clinic |
| Latest Activity | Most recent activity type logged |
| Activity Date | Date of the latest activity |
| Next Step | The current next step planned for this account |

### Sorting
Click any **column header** to sort by that column. Click again to reverse the sort direction. An ▲ or ▼ arrow shows the current sort.

### Searching
Type into the **search box** to filter by name, city, state, clinic, specialty, or NPI. Click **Apply**.

### Filtering
Use the dropdowns to narrow results:
- **State** — show only providers in one state
- **KAM** — show only Mark Franzen's or Mike Grillo's accounts

Click **Apply** to apply filters, or **Reset** to clear everything.

### Exporting to Excel
Click the orange **Export Excel** button. The download includes:
- **Providers tab** — all currently visible providers (your filters are applied)
- **Activities tab** — full activity log with who logged each one and when

---

## 3. The Provider Detail Page

Click any row on the dashboard to open that physician's detail page.

### What you'll see
- **Top card** — physician info: NPI, Specialty, Patient Focus, Conditions, Claims, Beneficiaries, City, State, Clinic Name, KAM, IR
- **Editable fields:**
  - **KAM** — change the Key Account Manager if needed (type a name and click Save)
  - **IR** — Inside Rep / additional info (free text — type and click Save)
- **Clinics** — every clinic location associated with this physician
- **Log Activity** — form to record a new activity (see below)
- **Activity History** — every activity ever logged for this physician

---

## 4. Logging an Activity

This is the main thing you'll do day-to-day.

1. Open the physician's detail page.
2. Scroll to the **Log Activity** card (green border).
3. Fill in:
   - **Activity Type** — choose one: `call`, `email`, `site visit`, `training`, `demo`
   - **Activity Date** — pick the date the activity happened
   - **Notes** — short description of what happened (free text)
   - **Next Step** — what should happen next with this account (free text). When you press **Add**, the Next Step is saved to the dashboard and the field is cleared so you can enter a fresh one next time.
4. Click **Add**.

Your name (set in the header) and the timestamp are automatically attached.

---

## 5. Editing a Past Activity

If you made a mistake or need to update an activity:

1. On the provider detail page, scroll to **Activity History**.
2. Click **Edit** in the row you want to change.
3. Update the type, date, notes, or next step.
4. Click **Save**.

The activity will show an **orange "edited"** tag with your name and the date you edited it. The original logger's name is still preserved — only the edit timestamp changes.

---

## 6. Quick Reference — Best Practices

- **Always set your name** in the header before working in the portal.
- **Log activities promptly** — same day if possible — so the dashboard always reflects reality.
- **Use clear notes** — future-you (and your colleagues) need to understand context: "Met with Dr. Smith and his MA Sarah; interested in trial program, sent reimbursement guide" beats "good meeting".
- **Update Next Step every time you log an activity** — that field drives what shows up on the dashboard so the team knows what to do next.
- **Export Excel** before big team meetings or QBRs to share a snapshot.

---

## 7. Troubleshooting

| Problem | Fix |
|---|---|
| "Set your name first" error | Type your name in the header and click Save |
| Activity didn't save | Make sure both Activity Type and Activity Date are filled in |
| Can't find a physician | Try searching by NPI or last name only; clear State/KAM filters |
| Page won't load | Check your internet; if it's still down, contact your admin |
